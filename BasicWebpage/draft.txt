Part 01:
- HTML tag is the parent/root of an HTML Document
- Head and Body are child of html root
- Data that should not be displayed in the webpage, should be under head container (Not rendered)
- Data that we want to display should be under body container

- HTML is not case sensitive; html and HTML is same

Part 02:
- HTML Attributes:
    #. Attributes are used to add more information to the tag
    #. Example: <html lang="en"></html>

- Heading tags should use to show the importance not for size

- Anchor Tag: it adds links to webpage
    #. Relative tag
    #. Absolute tag

- Image tag: use to add images to the webpage
    # src
    # alt 
    # width
    # height
    # better to use one between height or width to get proper dimension

- <br> tag: used to go nextline

- bold, italic, underline tags

- big and small tag that are used for big/small text in the webpage

- <hr> tag create sections with a line

-subscript and superscript tag

- Difference between pre and p tag:
    # pre considers the spaces , nextlines and tabs but p tag does not consider those
